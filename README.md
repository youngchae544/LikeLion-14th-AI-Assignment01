# LikeLion-14th-Assignment-Template
🦁 SKHU 멋쟁이사자처럼 14기 과제 PR 템플릿 레포지토리입니다.
1. 선택한 서비스 : 넷플릭스
2. 이 서비스는 무엇을 예측하는지 : 넷플릭스는 사용자가 어떤 콘텐츠(영화, 드라마 등등)를 좋아할지 개인 맞춤형 추천 시스템을 통해 사용자가 시청할 가능성이 높은 콘텐츠를 예측한다.
3. 어떤 data가 사용될까? :
   1) 사용자의 시청 기록(어떤 콘텐츠를 언제, 얼마나 봤는지에 대한 기록)
   2) 콘텐츠에 대한 평가(좋아요, 싫어요 등)
   3) 검색 기록 및 클릭 행동
   4) 사용자 정보(연령대, 지역, 성별 등)
   5) 콘텐츠 정보(장르, 배우, 감독, 키워드 등)
4. 모델은 어떻게 학습할까? : 넷플릭스는 크게 3가지의 머신러닝 학습 방식을 활용한다. 
   1) "협업 필터링(Collaborative Filtering)" : 나와 비슷한 취향을 가진 사람들이 좋아한 콘텐츠를 추천하는 방법이다.
   2) "콘텐츠 기반 필터링(Content-Based Filtering)" : 내가 이전에 좋아했던 콘텐츠와 비슷한 요소(장르, 배우, 감독 등)를 가진 콘텐츠를
   추천하는 방법이다.
   3) "딥러닝 기반 모델 " : 사용자 행동 패턴을 더 정교하게 분석하여 추천 정확도를 향상시키는 방법이다.
5. 이 모델이 더 좋아지려면 어떻게 해야 할까?
   1) 데이터 개선 : 더 다양한 사용자 데이터 확보하고 최신 데이터를 반영해야 한다. 
   2) 사용자 피드백 활용 : 좋아요/싫어요 등의 반응을 반영하여 추천 정확도를 향상 시켜야 한다.
   3) 모델 성능 향상 : 더 정교한 알고리즘 및 딥러닝 모델을 적용시켜야 한다. 이를 통해 복잡한 패턴까지 학습하여 추천 품질을 개선시킬 수 있다.
